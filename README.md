# 2023 MCM_A

2023 年美赛 A 题仓库（M 奖，$\leq 7\%$），征得队员们同意，将模拟代码、绘图、$\LaTeX$ 代码开源，供之后参加美赛的选手们参考。

## 使用方法

使用能编译 $\TeX$ 软件或者网站直接编译 `main.tex` 即可，图片和模拟代码资源使用相对路径，可直接在 `main.tex` 中修改。
